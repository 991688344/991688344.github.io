  1. win+r 打开regedit
  2. 进入HKEY_CLASSES_ROOT 中的 .md (MarkDown文件实例，添加其他格式的文件就找其他文件的后缀)
  3. 右击.md 新建项 ShellNew
  4. 点击ShellNew，右边空白处新建字符串值，名为NullFile
