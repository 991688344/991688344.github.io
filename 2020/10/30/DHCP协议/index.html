дёҖгҖҒDHCPз®Җд»Ӣ

1гҖҒдә§з”ҹиғҢжҷҜпјҡзҪ‘з»ңеўһеӨ§пјҢжүӢе·Ҙй…ҚзҪ®еӯҳеңЁеҫҲеӨҡй—®йўҳгҖҗдәәе‘ҳзҙ иҙЁиҰҒжұӮй«ҳгҖҒе®№жҳ“еҮәй”ҷгҖҒзҒөжҙ»жҖ§е·®гҖҒIPең°еқҖиө„жәҗеҲ©з”ЁзҺҮдҪҺгҖҒе·ҘдҪңйҮҸеӨ§пјҢдёҚеҲ©дәҺз®ЎзҗҶзӯүгҖ‘
2гҖҒDHCPзӣёеҜ№дәҺйқҷжҖҒжүӢе·Ҙй…ҚзҪ®зҡ„дјҳзӮ№гҖҗж•ҲзҺҮй«ҳгҖҒзҒөжҙ»жҖ§ејәгҖҒжҳ“дәҺз®ЎзҗҶзӯүгҖ‘

дәҢгҖҒDHCPзҡ„еҺҹзҗҶдёҺй…ҚзҪ®

пјҲдёҖпјүгҖҒDHCPзҡ„еҹәжң¬е·ҘдҪңиҝҮзЁӢгҖҗеҸ‘зҺ°йҳ¶ж®өгҖҒжҸҗдҫӣйҳ¶ж®өгҖҒиҜ·жұӮйҳ¶ж®өгҖҒзЎ®и®Өйҳ¶ж®өгҖ‘еҰӮдёӢеӣҫпјҡ

гҖҗеҸ‘зҺ°йҳ¶ж®өгҖ‘пјҡ

вҖӢ еңЁеҸ‘зҺ°йҳ¶ж®өпјҢDHCPе®ўжҲ·з«Ҝдјҡд»Ҙе№ҝж’ӯзҡ„ж–№ејҸз»ҷиҮӘе·ұжүҖеңЁеңЁе№ҝж’ӯеҹҹдёӯзҡ„DHCPжңҚеҠЎеҷЁйғҪеҸ‘йҖҒдёҖдёӘDHCP Discover жҠҘж–ҮпјҢзӣ®зҡ„жҳҜеҜ»жүҫзҪ‘з»ңдёӯзҡ„DHCP ServerпјҢе№¶иЎЁзӨәиҮӘе·ұйңҖиҰҒиҺ·еҫ—дёҖдёӘIPең°еқҖ

вҖӢ жӯӨе№ҝж’ӯжҠҘж–ҮйҮҮз”Ёдј иҫ“еұӮзҡ„UDP 68еҸ·з«ҜеҸЈеҸ‘йҖҒпјҲе°ҒиЈ…зҡ„зӣ®зҡ„з«ҜеҸЈдёәUDP 68еҸ·з«ҜеҸЈпјүпјҢз»ҸиҝҮзҪ‘з»ңеұӮIPеҚҸи®®е°ҒиЈ…еҗҺпјҢжәҗIPең°еқҖдёә0.0.0.0пјҲеӣ дёәжӯӨж—¶иҝҳжІЎжңүеҲҶй…ҚIPең°еқҖпјүпјҢзӣ®зҡ„IPең°еқҖдёә255.255.255.255пјҲжңүйҷҗе№ҝж’ӯIPең°еқҖпјүгҖӮеӣ дёәжӯӨж—¶пјҢDHCPе®ўжҲ·з«ҜжІЎжңүеҲҶй…ҚеҲ°IPең°еқҖпјҢд№ҹдёҚзҹҘйҒ“DHCPжңҚеҠЎеҷЁжҲ–DHCPдёӯз»§зҡ„IPең°еқҖпјҢжүҖд»ҘеңЁDHCP DISCOVERжҠҘж–ҮдёӯCiaddrпјҲе®ўжҲ·з«ҜIPең°еқҖпјүгҖҒYiaddrпјҲиў«еҲҶй…Қзҡ„DHCPе®ўжҲ·з«ҜIPең°еқҖпјүгҖҒSiaddrпјҲдёӢдёҖдёӘдёәDHCPе®ўжҲ·з«ҜеҲҶй…ҚIPең°еқҖзҡ„DHCPжңҚеҠЎеҷЁең°еқҖпјүгҖҒGiaddrпјҲDHCPдёӯз»§IPең°еқҖпјүиҝҷеӣӣдёӘеӯ—ж®өеқҮдёә0.0.0.0пјҢеҰӮдёӢжүҖзӨәгҖӮеҸҰеӨ–пјҢд»ҺдёӯеҸҜд»ҘзңӢеҲ°пјҢеңЁCHADDRеӯ—ж®өе’ҢDHCPйҖүйЎ№дёӯClientIdentifierеӯ—ж®өдёӯйғҪж ҮиҜҶдәҶDHCPе®ўжҲ·з«ҜзҪ‘еҚЎMACең°еқҖDHCP:Client Ethernet Address (chaddr) =08002B2ED85E

гҖҗжҸҗдҫӣйҳ¶ж®өгҖ‘пјҡ

вҖӢ жҳҜзҪ‘з»ңдёӯзҡ„DHCP Serverе“Қеә”жүҖж”¶еҲ°зҡ„DHCP Discover жҠҘж–ҮпјҢжҠҠеҮҶеӨҮжҸҗдҫӣзҡ„IPең°еқҖжҗәеёҰеңЁDHCP Offer жҠҘж–ҮдёӯпјҢе№¶д»ҘеҚ•ж’ӯзҡ„ж–№ејҸеҸ‘йҖҒз»ҷDHCP е®ўжҲ·з«ҜпјҲжіЁж„ҸиҝҷйҮҢOffer жҠҘж–ҮдёӯжҗәеёҰзҡ„IPең°еқҖе®ўжҲ·з«ҜиҝҳдёҚиғҪзӣҙжҺҘдҪҝз”Ёпјү

гҖҗиҜ·жұӮйҳ¶ж®өгҖ‘пјҡ

вҖӢ DHCPе®ўжҲ·з«ҜеңЁж”¶еҲ°зҡ„жүҖжңүOffer жҠҘж–ҮдёӯйҖүжӢ©жҺҘеҸ—з¬¬дёҖдёӘеҲ°иҫҫе®ўжҲ·з«Ҝзҡ„OfferпјҲйҖҡеёёжғ…еҶөдёӢйҖүжӢ©з¬¬дёҖдёӘжҺҘж”¶еҲ°зҡ„Offerпјү пјҢе№¶еҗ‘зӣёеә”зҡ„DHCP Server д»Ҙе№ҝж’ӯзҡ„ж–№ејҸеҸ‘йҖҒDHCP Request жҠҘж–ҮпјҢиЎЁзӨәиҮӘе·ұж„ҝж„ҸжҺҘеҸ—иҜҘOffer пјҲеңЁиҝҷдёӘDHCP Request жҠҘж–ҮдёӯжҗәеёҰжңүRдёҠзҡ„DHCP Server зҡ„ж ҮиҜҶвҖ”з§°дёәServer IdentifierпјҢиЎЁзӨәDHCP Client еҸӘж„ҝж„ҸжҺҘеҸ—RдёҠзҡ„OfferпјҢRд№ҹжҳҜйҖҡиҝҮеҲҶжһҗиҝҷдёӘж ҮиҜҶжқҘзЎ®е®ҡDHCP Client еҸӘж„ҝж„ҸжҺҘеҸ—иҮӘе·ұзҡ„Offer
иҖҢиҜҘдәҢеұӮе№ҝж’ӯеҹҹдёӯе…¶д»–зҡ„DHCPжңҚеҠЎеҷЁд№ҹдјҡж”¶еҲ°DHCP Client жүҖеҸ‘еҮәзҡ„DHCP Request жҠҘж–ҮпјҢе®ғд»¬еҲҷйҖҡиҝҮеҲҶжһҗиҝҷдёӘж ҮиҜҶжқҘзЎ®е®ҡDHCP Client е·Із»ҸжӢ’з»қжҺҘеҸ—иҮӘе·ұзҡ„Offerпјү

вҖӢ еңЁDHCP requestжҠҘж–Үе°ҒиЈ…зҡ„IPеҚҸи®®еӨҙйғЁдёӯпјҢе®ўжҲ·з«Ҝзҡ„ Source Addressд»Қз„¶жҳҜ 0.0.0.0пјҢж•°жҚ®еҢ…зҡ„ Destination д»Қз„¶жҳҜ 255.255.255.255гҖӮдҪҶеңЁDHCP requestжҠҘж–ҮдёӯCiaddrгҖҒYiaddrгҖҒSiaddrгҖҒGiaddr еӯ—ж®өзҡ„ең°еқҖеқҮ0.0.0.0

гҖҗзЎ®и®Өйҳ¶ж®өгҖ‘пјҡ

вҖӢ еҰӮжһңеңЁжҸҗдҫӣйҳ¶ж®өжүҖжҗәеёҰзҡ„IPең°еқҖиғҪжӯЈеёёең°еҲҶй…Қз»ҷDHCP Client дҪҝз”Ёзҡ„иҜқпјҢйӮЈд№ҲRдјҡд»ҘеҚ•ж’ӯзҡ„ж–№ејҸз»ҷDHCP Client еҸ‘йҖҒдёҖдёӘDHCP Ack жҠҘж–ҮпјҢжӯӨж—¶DHCP Client иҺ·еҸ–ең°еқҖжҲҗеҠҹпјӣдҪҶжҳҜеҰӮжһңеӣ дёәдёҖдәӣеҺҹеӣ пјҢеңЁжҸҗдҫӣйҳ¶ж®өжүҖжҗәеёҰзҡ„IPең°еқҖдёҚиғҪеҲҶй…Қз»ҷDHCP Client зҡ„иҜқпјҢиҝҷж—¶RеҲҷдјҡз»ҷDHCP Client еҸ‘йҖҒдёҖдёӘDHCP Nak жҠҘж–ҮпјҢиҖҢеҪ“DHCP Client ж”¶еҲ°Rзҡ„DHCP Nak жҠҘж–ҮеҗҺпјҢеҲҷиЎЁзӨәиҺ·еҸ–IPең°еқҖеӨұиҙҘпјҢйңҖиҰҒйҮҚж–°иҝӣе…ҘеҲ°еҸ‘зҺ°йҳ¶ж®өгҖӮ

еңЁDHCPе®ўжҲ·з«ҜеңЁиҺ·еҫ—дәҶдёҖдёӘIPең°еқҖд»ҘеҗҺпјҢе°ұеҸҜд»ҘеҸ‘йҖҒдёҖдёӘе…Қиҙ№ARPиҜ·жұӮжҺўжөӢзҪ‘з»ңдёӯжҳҜеҗҰиҝҳжңүе…¶е®ғдё»жңәдҪҝз”ЁIPең°еқҖпјҢжқҘйҒҝе…Қз”ұдәҺDHCPжңҚеҠЎеҷЁең°еқҖжұ йҮҚеҸ иҖҢеј•еҸ‘зҡ„IPеҶІзӘҒгҖӮд»ҘдёҠеӣӣдёӘйҳ¶ж®өеҰӮеӣҫ11-19жүҖзӨәпјҢзӣёеҪ“дәҺDHCPе®ўжҲ·з«ҜдёҺDHCPжңҚеҠЎеҷЁзҡ„еӣӣж¬ЎжҸЎжүӢиҝҮзЁӢгҖӮ

еҪ“DHCP Client жӯЈеёёиҺ·еҫ—IPең°еқҖеҗҺпјҢеҰӮдёӢеӣҫпјҡ

д»ҺDHCP Server иҺ·еҸ–зҡ„IPең°еқҖжңүдёҖдёӘз§ҹзәҰж—¶й—ҙпјҢDHCPеҚҸи®®и§„е®ҡз§ҹзәҰж—¶й—ҙзјәзңҒдёҚе°‘дәҺ1е°Ҹж—¶пјҢйҖҡеёёжғ…еҶөдёӢпјҢз§ҹзәҰж—¶й—ҙзјәзңҒеҖјдёә24е°Ҹж—¶гҖӮ
еңЁDHCPеҚҸи®®дёӯи§„е®ҡпјҢеңЁзјәзңҒжғ…еҶөдёӢпјҢеҪ“DHCP Client дҪҝз”ЁIPең°еқҖеҲ°иҫҫT1ж—¶пјҲз§ҹзәҰж—¶й—ҙзҡ„1/2пјүпјҢDHCP Clientдјҡд»ҘеҚ•ж’ӯзҡ„ж–№ејҸз»ҷRеҸ‘йҖҒдёҖдёӘDHCP Request жҠҘж–ҮпјҲиҜ·жұӮз»ӯз§ҹпјүпјҢеҪ“еҲ°иҫҫж—¶й—ҙT2пјҲз§ҹзәҰж—¶й—ҙзҡ„7/8пјҢеҚіз§ҹзәҰж—¶й—ҙзҡ„87.5%пјүд№ӢеүҚпјҢиӢҘDHCP Client ж”¶еҲ°Rзҡ„DHCP AckжҠҘж–ҮпјҢеҲҷз»ӯз§ҹжҲҗеҠҹпјҢйҮҚж–°ејҖе§Ӣз§ҹзәҰжңҹзҡ„и®Ўж—¶пјӣиӢҘжІЎжңүж”¶еҲ°пјҢеҲҷDHCP Client дјҡеңЁT2ж—¶еҲ»еҶҚж¬Ўд»Ҙе№ҝж’ӯзҡ„ж–№ејҸеҸ‘йҖҒDHCP Request жҠҘж–ҮпјҢиҜ·жұӮз»ӯз§ҹпјҢиҝҷж—¶еҰӮжһңDHCP Client еңЁз§ҹзәҰж—¶й—ҙеҲ°иҫҫд№ӢеүҚж”¶еҲ°Rзҡ„DHCP AckжҠҘж–ҮпјҢеҲҷз»ӯз§ҹжҲҗеҠҹпјҢйҮҚж–°ејҖе§Ӣз§ҹзәҰжңҹзҡ„и®Ўж—¶пјҢеҰӮжһңжІЎж”¶еҲ°пјҢеҲҷз»ӯз§ҹеӨұиҙҘпјҢDHCP Client дёҚиғҪз»§з»ӯдҪҝз”ЁиҝҷдёӘIPең°еқҖпјҢйңҖиҰҒйҮҚж–°д»ҺеҸ‘зҺ°йҳ¶ж®өејҖе§Ӣеҗ‘DHCPжңҚеҠЎеҷЁз”іиҜ·ж–°зҡ„IPең°еқҖ

пјҲдәҢпјүгҖҒDHCPдёӯз»§

1гҖҒDHCPдёӯз»§зҡ„дҪҝз”ЁеңәжҷҜ


д»ҺеүҚйқўDHCPзҡ„е·ҘдҪңеҺҹзҗҶжҲ‘д»¬еҸҜд»ҘзҹҘйҒ“пјҢDHCP Client еҝ…йЎ»е’ҢDHCP Server еңЁеҗҢдёҖдёӘдәҢеұӮе№ҝж’ӯеҹҹдёӯпјҢжүҚиғҪжҺҘж”¶еҲ°еҪјжӯӨеҸ‘йҖҒзҡ„DHCPжҠҘж–ҮпјҢDHCPжҠҘж–ҮжҳҜж— жі•и·Ёи¶ҠдәҢеұӮе№ҝж’ӯеҹҹдј ж’ӯзҡ„гҖӮиҝҷж—¶пјҢжҲ‘д»¬еҸҜд»ҘжңүдёӨз§Қж–№жі•и§ЈеҶіиҝҷдёӘй—®йўҳпјҡ
пјҲ1пјүгҖҒеңЁзҪ‘з»ңдёӯзҡ„жҜҸдёӘзҪ‘ж®өйғҪи®ҫзҪ®дёҖеҸ°DHCP ServerпјҢдҪҶжҳҜиҝҷдёӘж–№жі•д»Јд»·иҫғеӨ§пјҢеҫҲдёҚз»ҸжөҺпјҢйҖ жҲҗиө„жәҗзҡ„еҫҲеӨ§жөӘиҙ№пјҢжүҖд»ҘзҺ°е®һдёӯдёҖиҲ¬дёҚжҺЁиҚҗжӯӨж–№жі•
пјҲ2пјүгҖҒеҸӘи®ҫзҪ®дёҖдёӘDHCP ServerпјҢеҲ©з”ЁDHCPдёӯз»§жҠҖжңҜDHCP Relay жқҘе®һзҺ°дёҖдёӘDHCP Server еҗҢж—¶дёәеӨҡдёӘдәҢеұӮе№ҝж’ӯеҹҹдёӯзҡ„DHCP Client жңҚеҠЎпјҢиҝҷж ·ж—ўиҠӮзңҒжҲҗжң¬пјҢеҸҲдҫҝдәҺйӣҶдёӯз®ЎзҗҶ

2гҖҒDHCPдёӯз»§зҡ„е·ҘдҪңеҺҹзҗҶеҰӮдёӢеӣҫпјҡ


еңЁDHCPдёӯз»§зҡ„е·ҘдҪңеҺҹзҗҶдёӯе’ҢеүҚйқўзҡ„DHCPе·ҘдҪңеҺҹзҗҶеҹәжң¬зӣёеҗҢпјҢеҸӘжҳҜдёӯй—ҙDHCP Relay иҙҹиҙЈDHCP Client е’ҢDHCP Server д№Ӣй—ҙзҡ„DHCPжҠҘж–Үзҡ„иҪ¬еҸ‘пјҢиҝҷйҮҢиҰҒжіЁж„Ҹзҡ„жҳҜпјҡDHCP Relay дёҺDHCP Client д№Ӣй—ҙжҳҜд»Ҙе№ҝж’ӯзҡ„ж–№ејҸиҝӣиЎҢдәӨжҚўDHCPжҠҘж–Үзҡ„пјҢиҖҢDHCP Relay дёҺDHCP Server д№Ӣй—ҙжҳҜд»ҘеҚ•ж’ӯзҡ„ж–№ејҸжқҘдәӨжҚўDHCPжҠҘж–Үзҡ„гҖӮ

dhcpзҡ„е®ўжҲ·жңәзҡ„ipйҖүжӢ©зҡ„й—®йўҳпјҡ

вҖӢ йҰ–е…Ҳдјҡз”іиҜ·дёҠж¬ЎжҺҘе…ҘзҪ‘з»ңзҡ„ipең°еқҖпјҢеҰӮжһңиҝҷдёӘipең°еқҖжңӘиў«еҚ з”ЁиҖҢдё”жҳҜеӨ„дәҺж”№dhcpзҡ„зҪ‘ж®өйҮҢйқўзҡ„ipең°еқҖпјҢе°ұдјҡдјҳе…ҲйҖүжӢ©иҝҷдёҖдёӘipең°еқҖпјҢеҰӮжһңжІЎжңүпјҢйӮЈд№ҲжҲ‘д»¬е°ұдјҡеңЁиҢғеӣҙеҶ…д»Һе°ҸеҲ°еӨ§зҡ„йҖүжӢ©гҖӮ

DHCPе®ўжҲ·жңәзҡ„ipең°еқҖзҡ„з§ҹжңҹй—®йўҳпјҡ
