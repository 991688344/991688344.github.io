方法一

  • 1、停止容器(docker stop d00254ce3af7)

  • 2、停止docker服务(systemctl stop docker)

  • 3、修改这个容器的hostconfig.json文件中的端口（原帖有人提到，如果config.v2.json里面也记录了端口，也要修改）

  • 4、启动docker服务(systemctl start docker)

  • 5、启动容器(docker start d00254ce3af7)
